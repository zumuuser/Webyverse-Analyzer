info Overview

Webyverse provides comprehensive analysis tools for domains, email addresses, and phone numbers. Our platform evaluates various aspects to deliver actionable insights, helping you improve your online presence, security, and communication strategies.

domain Domain Analyzer

Evaluates websites across SEO, security, performance, hosting, and technical aspects to provide a holistic health score.

verified_user Email Verification

Validates email addresses with checks for format, domain, MX records, SMTP, and deliverability to ensure accuracy.

phone Phone Lookup

Provides detailed information about phone numbers, including carrier, location, line type, and potential risk factors.

Getting Started: Simply navigate to the respective tool (Domain, Email, or Phone) and enter the details you wish to analyze. Our system will automatically process and present detailed results.

score Scoring System

Our scoring system provides clear, actionable insights into your website's performance across various dimensions. Each score ranges from 0-100, with specific thresholds indicating different performance levels.

bar_chart Score Ranges Explained

90-100 Excellent
Outstanding performance with industry best practices fully implemented. Your asset excels in this area.
70-89 Good
Solid performance with most best practices in place. Minor optimizations could push you to excellent.
50-69 Average
Acceptable performance but significant improvements are needed to compete effectively.
0-49 Poor
Critical issues present that need immediate attention. This may impact user experience and rankings.

calculate How Scores Are Calculated

Overall Score Formula (Domain Analyzer)
Overall Score = (SEO Score + Security Score + Performance Score) ÷ 3

The overall score for domains is the average of its three individual scores, providing a balanced view of your website's health. Email and Phone scores are calculated based on a weighted sum of their respective validation checks and risk factors.

Important: Scores are calculated in real-time based on current data. Regular monitoring is recommended as changes can affect scores.

verified_user Email Verification

Our email verification tool performs comprehensive checks to ensure the validity and deliverability of an email address, helping you maintain clean mailing lists and improve communication.

check_circle_outline Validation Checks

Check Description Impact
Format Validation Ensures the email adheres to RFC standards (e.g., user@domain.com). Critical
Domain Verification Checks if the domain exists and has valid MX records. High
SMTP Testing Connects to the mail server to confirm it accepts mail for the address. High
Deliverability Analysis Determines if the email is likely to reach the inbox. High

star_half Email Reputation

Each email verification provides a reputation score, indicating the overall trustworthiness and deliverability likelihood of the email address.

90-100 Very Low Risk
Highly deliverable, excellent reputation.
0-49 Very High Risk
Likely undeliverable or spam trap.

bug_report Common Email Issues

Invalid Format: The email address does not follow standard email syntax.
No MX Records: The domain exists but has no mail exchange records, meaning it cannot receive emails.
SMTP Connection Failed: Unable to connect to the mail server, possibly due to temporary issues or a non-existent server.

phone Phone Lookup

Our phone lookup tool provides detailed information about phone numbers, helping you identify the caller, assess risk, and understand the number's characteristics.

data_usage Key Data Points

Data Point Description Use Case
Line Type Identifies if it's a mobile, landline, VoIP, or toll-free number. Filtering for marketing campaigns, fraud detection.
Carrier Information Details about the telecommunications provider. Understanding network reach, potential service issues.
Geographic Location Associated city, state, and country (for landlines/some mobiles). Localizing services, verifying user location.

network_check Carrier Information

Knowing the carrier can be vital for understanding call routing, potential service quality, and identifying specific network-related issues.

Why Carrier Info Matters

Different carriers may have varying service quality or be associated with specific types of numbers (e.g., some VoIP providers are more common for spam calls).

gavel Risk Assessment

We provide a risk assessment for phone numbers based on various factors, including known spam reports, line type, and historical data.

High Risk: Number associated with known spam, fraud, or suspicious activity.
Medium Risk: Number has some flags, but not definitively malicious (e.g., new VoIP number).
Low Risk: Number appears legitimate and has no negative flags.

build General Troubleshooting

Common issues you might encounter when using Webyverse tools and how to resolve them.

error_outline Analysis Problems expand_more

cloud_off "Unable to fetch" or "Not Found" Error

Possible Causes:

  • Input (domain, email, phone) is misspelled or doesn't exist.
  • The target website/service is down or unreachable.
  • Firewall or security measures blocking our analysis servers.

Solutions:

  1. Double-check the input for typos.
  2. Verify the target is accessible in your browser or via other means.
  3. Wait a few minutes and try again.

incomplete_circle Incomplete Results

Common Reasons:

  • Target blocks automated requests or has very slow response times.
  • Missing or malformed data on the target (e.g., missing meta tags).

Solutions:

  1. Ensure the target loads quickly and completely in a browser.
  2. Check for proper implementation of required data (e.g., meta tags for SEO).
trending_down Lower Than Expected Scores expand_more

search_off SEO Score

  • Missing or incomplete meta tags (title, description).
  • No Open Graph or Twitter Card implementation.
  • Missing XML sitemap.

lock_open Security Score

  • No SSL certificate or an expired one.
  • Missing crucial security headers (HSTS, CSP, X-Frame-Options).

timer_off Performance Score

  • No compression (Gzip/Brotli) enabled.
  • Missing or incorrect caching headers.
  • Slow or unreliable hosting provider.

api API Reference

For developers who want to integrate Webyverse analysis into their own applications, we provide API endpoints for programmatic access to our domain, email, and phone analysis data.

endpoint Available Endpoints

domain GET /api/domain/{name}

Retrieve detailed analysis for a specific domain.

GET /api/domain/example.com
verified_user GET /api/email/{address}

Get comprehensive verification results for an email address.

GET /api/email/user@example.com
phone GET /api/phone/{number}

Lookup details and risk assessment for a phone number.

GET /api/phone/+15551234567
code API Usage Examples expand_more

JavaScript/Fetch Example (Domain)

fetch('/api/domain/example.com') .then(response => response.json()) .then(data => { console.log('SEO Score:', data.seo_score); console.log('Security Score:', data.security_score); }) .catch(error => console.error('Error:', error));

Python Example (Email)

import requests response = requests.get('http://your-domain.com/api/email/user@example.com') if response.status_code == 200: data = response.json() print(f"Deliverability: {data['deliverability_status']}") print(f"Reputation Score: {data['reputation_score']}") else: print("Email not found or error occurred")
Rate Limiting: API endpoints are rate-limited to prevent abuse. Contact us for higher rate limits if needed for legitimate use cases.
